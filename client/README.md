# client

The LogLens frontend: **React + TypeScript + Vite + Tailwind CSS**.

> 🚧 Scaffolded in **Phase 6** of the roadmap. For now this folder is a
> placeholder so the repository structure is complete.

Planned screens:

- **Home / Input** — paste a log or upload a `.txt` / `.log` file
- **Report** — the structured AI analysis (problem, severity, category, causes, debug steps)
- **History** — list of past analyses
- **Detail** — full report for a single analysis, with Markdown export and a copy-ready GitHub Issue
